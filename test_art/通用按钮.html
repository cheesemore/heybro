通用按钮

游戏向斜切矩形按钮 · 下方为可配置项，修改后点「应用到画布」。

可与描边同色或单独指定
留空则按高度自动
留空则约 0.4× 高度
留空则随宽高自动
可配置项说明（与代码 createGameGeneralButton 一致）
text
按钮文字。
width / height
按钮像素尺寸。
fill / stroke
填充与描边，支持 #RRGGBB 或 0xRRGGBB。
textColor
字色；不写则与描边同色。本页用勾选「跟随描边」实现。
strokeWidth / fontSize / cut
可选；不写时按高度、宽度自动估算。
fontWeight
默认 bold。
fontFamily
CSS 字体列表字符串。
运行时
代码里可对实例调用 btn.redraw({ ... }) 局部更新。
