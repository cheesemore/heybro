# 通用按钮.html · 使用说明

对应页面：`通用按钮.html`（与本 `.md` 同名，方便一起找）。

## 这是什么

**PixiJS 7** 游戏向 **斜切通用按钮**：页面顶部有 **表单** 可改文案、配色、宽高、画布背景等；中间画布实时预览。依赖 **jsdelivr CDN** 上的 `pixi.js@7.4.2`，**需要联网**。

## 怎么打开

1. 联网后 **双击** `通用按钮.html`。
2. **推荐**：在本目录起静态服务（中文文件名在地址栏可自动编码）：

```bash
cd /d E:\cursor\heybro\test_art
python -m http.server 8765
```

浏览器打开：`http://127.0.0.1:8765/通用按钮.html`  
若地址栏不好输入中文，可复制资源管理器里的路径，或使用编码后的 URL。

## 怎么用到项目里

- 页面内联脚本暴露 **`createGameGeneralButton(options)`** 与 **`toPixiColor`**。
- 创建：`var b = createGameGeneralButton({ text: '确定', width: 224, height: 60, fill: '#EBE5A7', stroke: '#3D3D33', ... })`，再 `stage.addChild(b)`。
- 运行时更新：`b.redraw({ text: '继续', fill: '#c8e6c9', width: 280 })`；仅改宽高时会自动重算切角等（见脚本内 `redraw` 逻辑）。
- 集成到打包项目时：把相关函数拷入 `*.js` / `*.ts`，改为 `import * as PIXI from 'pixi.js'` 或保持 CDN，按团队规范处理。

## 小提示

改表单或脚本后 **`Ctrl+F5`** 强刷。本页不是 npm 包，无 `package.json`。
